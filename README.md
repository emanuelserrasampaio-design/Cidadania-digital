# Cidadania Digital e Inteligência Artificial: O Impacto das Deepfakes

## 🎯 Objetivo do Projeto
Este site foi desenvolvido como atividade de Recuperação Prática para o componente de Educação Digital e IA. O objetivo principal é conscientizar a comunidade escolar sobre os perigos das mídias manipuladas (deepfakes) e ensinar boas práticas para identificar notícias falsas no ambiente digital.

## 🛠️ Tecnologias Utilizadas
- HTML5 (Estrutura Semântica)
- CSS3 (Layout Responsivo com Flexbox)
- JavaScript Vanilla (Manipulação do DOM e Lógica Interativa)
